What is Sygue?

Sygue is a programming language that strives to be better and more effective than C/C++. It will have a number of features that C/C++ don't possess such as dynamic arrays and a built-in build system. It will also be without several "features" that C/C++ possess such as removing '->' for accessing struct members from pointers an classes.
